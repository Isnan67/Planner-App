<h1 align="center">Planner App</h1>
<p align="center"><i>Aplikasi produktivitas all-in-one untuk mengatur jadwal, catatan, dan keuangan.</i></p>

---

**Planner** adalah aplikasi produktivitas yang dirancang untuk membantu Anda mengatur jadwal, mencatat hal penting, dan mengelola keuangan pribadi dalam satu tempat yang praktis!

### ✨ Fitur Unggulan Planner:

- 📅 **Manajemen Jadwal & Tugas:** Buat to-do list harian Anda. Dilengkapi dengan sistem Alarm akurat yang akan mengingatkan Anda tepat waktu!
- 📝 **Catatan Pintar:** Simpan ide dan momen penting. Mendukung penambahan foto/gambar langsung dari galeri HP Anda ke dalam catatan.
- 💰 **Manajer Keuangan:** Catat pemasukan dan pengeluaran dengan mudah. Dilengkapi dengan Diagram Keuangan interaktif dan pemilih bulan.
- 🎨 **Personalisasi Tema:** Dukungan penuh untuk Mode Gelap (Dark Mode) dan berbagai pilihan warna tema sesuai selera Anda.
- 🔥 **Sistem Streak:** Buka aplikasi berturut-turut untuk mendapatkan reward bebas iklan!
- 🧹 **Manajemen Data:** Fitur Wipe/Reset data untuk menghapus jadwal, catatan, atau laporan keuangan secara instan.

---

### 📥 Cara Download & Install

Untuk mencoba aplikasi ini, silakan ikuti langkah berikut:
1. Masuk ke halaman **[Releases](../../releases/latest)**.
2. Download file **`app-release.apk`** yang tersedia di bagian *Assets*.
3. Buka file APK tersebut di HP Android Anda dan lakukan instalasi.

*Dibuat khusus untuk memudahkan produktivitas harian Anda. Download sekarang dan mulai atur hidupmu jadi lebih rapi!*
